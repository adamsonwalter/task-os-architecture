## Value Proposition Frameworks

### Framework 1: The 3-Step Process
Show simplicity (↓ Effort, ↓ Time Delay)

```
1. [Simple Action] — "Sign up in 60 seconds"
2. [Simple Action] — "Connect your data"
3. [Outcome] — "Watch your pipeline grow"
```

### Framework 2: Problem → Agitate → Solve (PAS)
For problem-aware audiences

```
The Problem: [State their pain vividly]
Why It Hurts: [Agitate consequences]  
The Solution: [Your mechanism]
```

### Framework 3: Hormozi Value Stack
Maximize perceived value

```
[Core Offer] — Worth $X
+ [Bonus 1] — Worth $Y  
+ [Bonus 2] — Worth $Z
+ [Bonus 3] — Worth $W
───────────────────────
Total Value: $[Sum]
Your Price: $[Much Less]
```

### Framework 4: Comparison Table
For solution-aware audiences

| Without [Product] | With [Product] |
|-------------------|----------------|
| [Pain point 1] | [Benefit 1] |
| [Pain point 2] | [Benefit 2] |
| [Pain point 3] | [Benefit 3] |

## The Four Levers Articulation

Map your value proposition copy to each lever:

### Dream Outcome Statements
```
"Finally [achieve goal] you've always wanted"
"Imagine [vivid description of success state]"
"The [result] that changes everything"
```

### Likelihood Boosters
```
"Proven by [X] customers" 
"The exact system that generated [$Y] for [Z]"
"No guesswork—follow the framework"
```

### Time Delay Reducers  
```
"See results in [timeframe]"
"Get started in [minutes]"
"Your first [outcome] by [when]"
```

### Effort Minimizers
```
"Done for you [X]"
"No [technical thing] required"
"Just [simple action], we handle the rest"
"Works while you sleep"
```

## Validation Checklist

☐ Does copy address all four Hormozi levers?  
☐ Is the mechanism (how it works) clear?  
☐ Is jargon eliminated or explained?  
☐ Would a skeptic understand the value?  
☐ Is there a secondary CTA at section end?
