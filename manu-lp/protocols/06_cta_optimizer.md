# Phase 6: CTA Optimizer Protocol

**Purpose**: Design and place CTAs for maximum conversion  
**Value Equation Focus**: Effort (↓↓↓) through friction elimination

---

## CTA Hierarchy

**Rule**: One primary CTA, one goal, multiple placements.

```
PRIMARY CTA: The main action (e.g., "Start Free Trial")
  ↓
SECONDARY CTA: Lower-commitment alternative (e.g., "Watch Demo")
  ↓  
MICRO-CTA: Inline links/buttons for specific segments
```

---


**Rule**: CTA should be visible without scrolling, and repeated at natural scroll-stop points.

## CTA Button Copy Formulas

### Formula 1: Action + Outcome
```
[Verb] + [Desired Outcome]
Examples:
- "Get More Leads"
- "Start Growing Revenue"
- "Build My Landing Page"
```

### Formula 2: Possessive + Object
```
Get My [Thing]
Examples:
- "Get My Free Report"
- "Claim My Discount"
- "Start My Free Trial"
```

### Formula 3: Low-Friction Action
```
[See/Learn/Discover] + [Benefit]
Examples:
- "See How It Works"
- "Watch the Demo"
- "Explore Features"
```

### Formula 4: Urgency Verbs
```
[Claim/Reserve/Lock In] + [Offer]
Examples:
- "Claim Your Spot"
- "Reserve My Seat"
- "Lock In This Price"
```

## CTA Button Design Rules

| Element | Best Practice |
|---------|--------------|
| **Color** | High contrast with background, not brand color if it blends |
| **Size** | Large enough to tap on mobile (min 44x44px touch target) |
| **Shape** | Rounded corners (softer, more inviting) |
| **Text** | 2-5 words, sentence case or Title Case |
| **Whitespace** | Generous padding inside and around |
| **Shadow** | Subtle elevation helps it "pop" |

### A/B Test Priorities
1. Button copy (highest impact)
2. Button color
3. Button placement
4. Supporting micro-copy

## Micro-Copy Patterns (Below CTA)

**Purpose**: Reduce final-moment friction

| Concern | Micro-Copy Solution |
|---------|---------------------|
| Money risk | "No credit card required" |
| Time concern | "Takes 2 minutes" |
| Commitment fear | "Cancel anytime" |
| Trust worry | "100% secure checkout" |
| Effort anxiety | "No technical skills needed" |
| Social proof | "Join 10,000+ users" |
| Guarantee | "30-day money-back guarantee" |

**Format**: Small text, checkmarks or bullets, directly below CTA

---

## Secondary CTA Strategy

**When to use**: When visitors aren't ready for primary action

| If Primary Is | Secondary Could Be |
|---------------|-------------------|
| Start Free Trial | Watch 2-Min Demo |
| Buy Now | Learn More |
| Book a Call | Download Free Guide |
| Sign Up | See Pricing |
| Get Quote | View Case Studies |

**Placement**: Below primary CTA, smaller/lighter styling

---

## Form Optimization (If CTA Is Form)

### Field Reduction Rules
| Fields | Expected Drop |
|--------|---------------|
| 1 (email only) | Baseline |
| 2 (name + email) | -10% |
| 3+ fields | -25% per additional field |

**Principle**: Ask only for what you absolutely need NOW.

### Form UX Checklist
- ☐ Single column layout
- ☐ Field labels above inputs (not inside)
- ☐ Auto-focus on first field
- ☐ Mobile keyboard optimization (email keyboard for email)
- ☐ Clear error states
- ☐ Progress indicator (if multi-step)
