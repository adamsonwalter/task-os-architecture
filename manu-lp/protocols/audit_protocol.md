# Audit Protocol

**Purpose**: Systematically evaluate existing landing pages against conversion best practices  
**When to Use**: User provides URL or screenshots of existing page

---

## Audit Framework: CLEAR

| Element | Question | Score Range |
|---------|----------|-------------|
| **C**larity | Can a visitor explain the offer in one sentence? | 0-20 |
| **L**everage (Emotion) | Does copy trigger desire or solve pain? | 0-20 |
| **E**vidence (Trust) | Is there credible social proof? | 0-20 |
| **A**ction (CTA) | Is next step obvious and frictionless? | 0-20 |
| **R**elevance | Does page speak to target audience specifically? | 0-20 |

**Total Score**: /100
