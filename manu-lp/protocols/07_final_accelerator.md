## Urgency Types (Ethical)

### 1. Deadline-Based (Time Urgency)
```
"Offer ends [date/time]"
"Enrollment closes in 3 days"
"Limited-time pricing expires Friday"
```
**Best for**: Product launches, cohort programs, sales events

### 2. Scarcity-Based (Quantity Limits)
```
"Only 7 spots remaining"
"Limited to 50 beta users"
"97% sold — only 12 left"
```
**Best for**: High-touch services, exclusive offers, physical products

### 3. Consequence-Based (Cost of Inaction)
```
"Every day without [solution] costs you [loss]"
"Your competitors aren't waiting"
"The problem isn't going away"
```
**Best for**: B2B, high-consideration purchases

### 4. Momentum-Based (FOMO)
```
"Join 10,000+ who already started"
"347 people signed up this week"
"Don't miss what everyone's talking about"
```
**Best for**: Community-driven products, trending offers

## Sticky CTA Options

### Sticky Bottom Bar
```
┌─────────────────────────────────────────────────────────┐
│ [Brief offer reminder]        [PRIMARY CTA BUTTON]      │
└─────────────────────────────────────────────────────────┘
```
Appears after user scrolls past hero, stays fixed at bottom.

### Floating CTA
```
Small button in bottom-right corner, expands on hover.
Less aggressive than sticky bar.
```

### Exit Intent Popup
```
Triggered when mouse moves toward browser close.
Offers final incentive or lead capture.
```

## Countdown Timer Best Practices

**When to use**: Real deadlines only (ethical urgency)

| ✅ Do | ❌ Don't |
|-------|---------|
| Use for actual sales events | Create fake scarcity |
| Display timezone context | Reset timer on refresh |
| Honor the deadline | Keep extending indefinitely |
| Pair with bonus expiration | Use evergreen fake countdowns |

---

## Exit Strategy (For Non-Converters)

If primary CTA is too high-commitment, offer softer off-ramp:

```
Not ready yet? [Softer alternative]

Examples:
- "Bookmark this page and come back when you're ready"
- "Get our free [resource] while you decide"
- "Have questions? [Chat/Email link]"
- "Subscribe for updates (no spam)"
```
