# Phase 1: Hero Forge Protocol

**Purpose**: Create above-the-fold hero section that stops scroll and triggers action  
**Value Equation Focus**: Dream Outcome (↑) + Time Delay (↓)

---

## Hero Section Anatomy

```
┌─────────────────────────────────────────────────────────┐
│ [Trust Signal Micro-Bar] "Trusted by 10,000+ customers" │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   HEADLINE: Dream Outcome Statement                     │
│   Subhead: How + Timeframe + Proof                     │
│                                                         │
│   [PRIMARY CTA BUTTON]                                  │
│   No credit card required • 2-minute setup             │
│                                                         │
│   [Hero Image/Video showing transformation or product]  │
│                                                         │
├─────────────────────────────────────────────────────────┤
│ [Logo Bar] "As featured in:" / "Trusted by:" logos     │
└─────────────────────────────────────────────────────────┘
```

**Rule**: One headline, one subhead, one CTA. Nothing more above the fold.

---

## Headline Formulas

### Formula 1: Outcome + Timeframe
```
"Get [Dream Outcome] in [Timeframe]"
Example: "Get 3x More Qualified Leads in 30 Days"
```

### Formula 2: Transformation Bridge
```
"From [Current State] to [Dream State] Without [Sacrifice]"
Example: "From Overwhelmed to Organized Without Hiring a VA"
```

### Formula 3: Specific Outcome
```
"[Specific Number] [Audience] Used This to [Achieve Result]"
Example: "10,247 Founders Used This to Double Their Revenue"
```

### Formula 4: Question Hook (Problem Aware)
```
"What If You Could [Dream Outcome]?"
Example: "What If You Could Close Deals Without Cold Calling?"
```

### Formula 5: Commanding (Most Aware)
```
"[Action Verb] Your [Desired Object] Today"
Example: "Launch Your Course Today—Get Paid Tomorrow"
```

---

## Subhead Patterns

Subhead answers: "How does this work and why should I believe you?"

| Pattern | Template |
|---------|----------|
| **Mechanism + Proof** | "Our [mechanism] has helped [X] [audience] achieve [result]" |
| **Speed + Ease** | "[Achieve result] in just [time] with [simple process]" |
| **Contrast** | "Without [thing they hate], without [sacrifice], without [risk]" |
| **Social Proof** | "Join [number] [audience] who already [achieved outcome]" |

---

## CTA Button Copy

**Bad**: Submit, Learn More, Click Here  
**Good**: Action verb + Outcome

| Template | Example |
|----------|---------|
| Get My [Benefit] | Get My Free Report |
| Start [Verb]-ing [Outcome] | Start Growing Revenue |
| [Verb] Now — [Benefit] | Join Now — Get Instant Access |
| Claim Your [Thing] | Claim Your Free Strategy Call |
| See How It Works | See How It Works |

---

## Micro-Copy (Below CTA)

Reduce perceived effort and risk:

```
✓ No credit card required
✓ Setup in 2 minutes  
✓ Cancel anytime
✓ 30-day money-back guarantee
✓ Free forever plan available
✓ Instant access
```

Choose 2-3 relevant to reduce friction.

---

## Trust Signal Micro-Bar

Options for top of hero:
- "Trusted by 10,000+ businesses worldwide"
- "★★★★★ 4.9/5 from 500+ reviews"
- "Featured in [Logo] [Logo] [Logo]"
- "Winner: Best [Category] 2025"

---

## Hero Image/Video Guidelines

| Type | When to Use |
|------|-------------|
| **Product screenshot** | SaaS, clear UI value |
| **Transformation visual** | Before/after, coaching, fitness |
| **Person using product** | Relatable, B2C |
| **Dashboard/results** | Analytics, performance tools |
| **Short video (<30s)** | Complex products, demos |

**Avoid**: Generic stock photos, cluttered images, abstract graphics

## Validation Checklist

☐ Can a visitor explain the offer in one sentence?  
☐ Is headline benefit-driven, not feature-driven?  
☐ Is there exactly ONE primary CTA?  
☐ Is the CTA visible without scrolling?  
☐ Does micro-copy reduce fear/friction?  
☐ Is there a trust signal above or near CTA?  
☐ Does hero work on mobile (tap-friendly CTA)?
